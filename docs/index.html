Module luastring

A stand-alone extension of string operations for Lua.

Written in pure Lua for excellent compatibility, maintainability, and guaranteed portability.

LuaString is an extension of the string library:

Usage:

    local string = require("luastring")

lfind (str, substr)
Find the index of the first occurance of substr inside of str. This is syntactic sugar for a plain-text search. It's unwise to use syntactic sugar if you're desperate for performance, as it'll add one function call.

Parameters:

  • str string The string to search.
  • substr string The substring to search str for.
rfind (str, substr)
Find the index of the last occurance of substr inside of str.

Parameters:

  • str string The string to search.
  • substr string The substring to search str for.

Returns:

    number This will return 0 if nothing was found, otherwise the starting index of the substring.

Usage:

    local str = "hello hello hello hello hello hello"
local lastidx = luastring.rfind(str, "hello")
assert(lastidx == 31) -- Lua indices advance us by 1.
assert(str:sub(lastidx) == "hello")

title (str)
Creates a new title-case string from str. There's no universal title-case specification, but this will capitalize the first letter of any word that is not: "of", "the", "at", excluding from the first word.

Parameters:

  • str string The string to convert.

Returns:

    string

Usage:

    luastring.title("the paranormal of the activity") == "The Paranormal of the Activity"

partition (str, delimiter)
Splits a string into two parts: one part before the first occurance of delimiter, and one after.

Parameters:

  • str string The string to partition.
  • delimiter string The delimiter to partition the string upon.

Returns:

    string,string or string,nil

Usage:

    local before, after = luastring.partition("hello.wor.ld", ".")
assert(before == "hello")
assert(after == "wor.ld")

local before, after = luastring.partition("doesn't contain", "!")
assert(before == "doesn't contain")
assert(after == nil)
rpartition (str, delimiter)
Splits a string into two parts: one part before the first right-side occurance of delimiter, and one after.

Parameters:

  • str string The string to partition.
  • delimiter string The delimiter to partition the string upon.

Returns:

    string,string or string,nil

Usage:

    local before, after = luastring.rpartition("hello?there?world", "?")
assert(before == "hello?there")
assert(after == "world")

assert(before == "hello?there?world")
assert(after == nil)
ljustify (str, character, len)
Inserts character into the left-side of the string until the string's length meets len. This doesn't do anything if the len exceeds the length of str.

Parameters:

  • str string The string to justify.
  • character string The single-length character to justify str with.
  • len number This should be set to the integeral length you expect str to be after justification.

Returns:

    string

Usage:

  • luastring.ljustify("hello", "*", 10) == "*****hello"
  • luastring.ljustify("hello", "*", 5) == "hello"

lrjustify (str, character, len)
Inserts character into both sides of the string until the string's length meets len. Since this is two-sided justification, the padding length will always round up. This doesn't do anything if the len exceeds the length of str.

Parameters:

  • str string The string to justify.
  • character string The single-length character to justify str with.
  • len number This should be set to the integeral length you expect str to be after justification.

Returns:

    string

Usage:

  • luastring.lrjustify("hello", "*", 10) == "***hello***"
  • luastring.lrjustify("hello", "*", 12) == "****hello****"
ltruncate (str, length, ellipsis)
Truncates a string from the left to meet the length specification. Optionally prepending an ellipsis to the truncated string.

Parameters:

  • str string The string to truncate, if needed.
  • length number An integral number describing the desired length of str.
  • ellipsis boolean or nil A boolean specifying whether or not to append an ellipsis to the truncated result.

Returns:

    string The string remains unchanged if it doesn't exceed length.

Usage:

  • luastring.ltruncate("Hello, World!", 5) == ", World!"
  • luastring.ltruncate("Hello, World!", 5, true) == "..., World!"
rtruncate (str, length, ellipsis)
Truncates a string from the right to meet the length specification. Optionally appending an ellipsis to the truncated string.

Parameters:

  • str string The string to truncate, if needed.
  • length number An integral number describing the desired length of str.
  • ellipsis boolean or nil A boolean specifying whether or not to append an ellipsis to the truncated result.

Returns:

    string The string remains unchanged if it doesn't exceed length.

Usage:

  • luastring.rtruncate("hello world", 5) == "hello"
  • luastring.rtruncate("hello world", 5, true) == "hello..."
translate (str, translation_table)
Translates str in accordance with the translation table you create. Translation tables are character maps. This loops over each character and requests your translation table for an alternate representation.

Parameters:

  • str string The string to translate.
  • translation_table table The translation table for this string. Example below.

Returns:

    string Returns the original string if no translation was made.

Usage:

    local translation_table = {
    [","] = "Z", ["!"] = "?", ["W"] = "R"
}
local translated = luastring.translate("Hello, World!", translation_table)
assert(translated == "HelloZ Rorld?")
