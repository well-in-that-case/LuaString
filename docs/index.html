Module luastring

An extension of string operations for Lua.

LuaString is:

  • Zero-dependency.
  • Permissive (via MIT).
  • A pure Lua implementation.
  • Fully compatible with ANSI C, LuaJIT, and Lua 5.1+.
  • Designed for maintainability, compatibility, and portability.

The fully-featured module is less than 6kb if you vouch for the minified file.

License

LuaString is licensed under MIT. View the terms here.

LuaString is an extension of the string library:

Usage:

    local string = require("luastring")

partition (str, substr)
Returns one part of the string before the first substr & one after.

Parameters:

  • str string The string to partition.
  • substr string The substring to partition the string upon.

Returns:

    string,string or string,nil

Usage:

    local before, after = luastring.partition("hello.wor.ld", ".")
assert(before == "hello")
assert(after == "wor.ld")

assert(before == "doesn't contain")
assert(after == nil)
rpartition (str, substr)
Returns one part of the string before the last substr & one after.

Parameters:

  • str string The string to partition.
  • substr string The substring to partition the string upon.

Returns:

    string,string or string,nil

Usage:

    local before, after = luastring.rpartition("hello?there?world", "?")
assert(before == "hello?there")
assert(after == "world")

local before, after = luastring.rpartition("hello?there?world", ".")
assert(before == "hello?there?world")
assert(after == nil)
expand (str, substitutes)
Position-agnostic, keyword-defined string formatting. Keyword format specifiers are defined with {this syntax}. If you pass a string as the substitutes argument, then it will serve as a cover-all substitute. If a lookup or callback returns nil, then the keyword is left unchanged.

Parameters:

  • str string The string to format.
  • substitutes table, string or function Table or callback to recieve each keyword.

Usage:

    -- Lookup table example.
local str1 = luastring.expand("{t1} {t2}", { t1 = "hello", t2 = "world" })
assert(str1 == "hello world")

-- Callback example.
local str2 = luastring.expand("{t1} {t2} {t3}", function (keyword)
    if keyword == "t1" then
        return "Keyword1"
    elseif keyword == "t2" then
        return "Keyword2"
    end
end)
assert(str2 == "Keyword1 Keyword2 {t3}")

-- String example.
local str3 = luastring.expand("{t1} {t2} {t3}", "abc")
assert(str3 == "abc abc abc")
subsequence (str, min, max)
Very similar to sequence, but allows you to specify a range.

Parameters:

  • str string The string to destruct.
  • min number or nil The minimum index to begin from. By default, it's 1.
  • max number or nil The maximum index to include. By default, it's #str.

Returns:

    table

Usage:

    for index, value in ipairs(luastring.subsequence("hello world", 7)) do
    print(index, value) -- 7 w ...
end
between_substr (str, substr1, substr2)
Returns the substring between substr1 & substr2.

Parameters:

  • str string The string to search.
  • substr1 string The first substring to start with.
  • substr2 string The last substring to end with.

Returns:

    string or nil Returns nil if nothing was found between the substrings.

Converts "1k" into "1000". This returns a float by default. It only supports thousands to quintillions ("k" to "q") currently.

Parameters:

  • str string The numeric string to convert.
  • floor boolean A boolean specifying whether or not to round-down the result.

Returns:

String Justification

ljustify (str, character, len)
Inserts char into the start of the string until the length meets len. This doesn't do anything if the len exceeds the length of str.

Parameters:

  • str string The string to justify.
  • character string The single-length character to justify str with.
  • len number This should be set to the integeral length you expect str to be after justification.

Returns:

    string

Usage:

    assert(luastring.ljustify("hello", "*", 10) == "*****hello")
assert(luastring.ljustify("hello", "*", 5) == "hello")
rjustify (str, character, len)
Inserts char into the end of the string until the length meets len. This doesn't do anything if the len exceeds the length of str.

Parameters:

  • str string The string to justify.
  • character string The single-length character to justify str with.
  • len number This should be set to the integeral length you expect str to be after justification.

Returns:

    string

Usage:

    assert(luastring.rjustify("hello", "*", 10) == "hello*****")
assert(luastring.rjustify("hello", "*", 5) == "hello")
lrjustify (str, character, len)
Inserts char into the start & end of the string until the length meets len. Since this is two-sided justification, the padding length will always round up. This doesn't do anything if the len exceeds the length of str.

Parameters:

  • str string The string to justify.
  • character string The single-length character to justify str with.
  • len number This should be set to the integeral length you expect str to be after justification.

Returns:

    string

Usage:

    assert(luastring.lrjustify("hello", "*", 10) == "***hello***")
assert(luastring.lrjustify("hello", "*", 12) == "****hello****")

Truncates a string from the left to meet the length specification. Optionally prepending an ellipsis to the truncated string.

translate (str, translation_table)
Translates str in accordance with the translation table you create. Translation tables are character maps. This loops over each character and requests your translation table for an alternate representation.

Parameters:

  • str string The string to translate.
  • translation_table table The translation table for this string. Example below.

Returns:

    string Returns the original string if no translation was made.

Usage:

    local translation_table = {
    [","] = "Z", ["!"] = "?", ["W"] = "R"
}
local translated = luastring.translate("Hello, World!", translation_table)
assert(translated == "HelloZ Rorld?")

Replaces several occurances of old inside str with new.

String Generation

randomstring (length, excluded_chars)
Generates a random string from printable ASCII characters. The ASCII printable range can be observed from the 32-126 byte range.

Parameters:

  • length number The desired length of your randomized string.
  • excluded_chars table A character map of characters to exclude from the randomized string. Example below.

Returns:

    string

Usage:

    local excluded_chars = {
    [" "] = 0, -- Exclude spaces.
    ["'"] = 0, -- Exclude single-width commas.
}
local randomized = luastring.randomstring(41, excluded_chars)
print(randomized) -- *@K@iuX}jN?atGfi|4?};wipr[~o6ORKHdou*OPZ8
